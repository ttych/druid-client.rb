## Cli Usage

The cli is `druidc`.

### Tasks

* druidc tasks
* druidc tasks list
* druidc tasks l


## Installation

Add this line to your application's Gemfile:

```ruby
gem 'druid-client'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install druid-client
